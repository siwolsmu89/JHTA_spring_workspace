@ModelAttribute("이름")
	- 메소드에 부착하는 어노테이션이다.
	- 이 어노테이션이 부착된 메소드가 반환하는 값을 Model 객체에 지정된 이름으로 저장한다.
	- 해당 컨트롤러에 정의된 요청이 올 때마다 이 어노테이션이 부착된 메소드가 실행된다.
	- 해당 컨트롤러와 관련된 모든 화면에서 Model에 저장된 정보를 사용할 수 있다.
	- 컨트롤러 내의 다른 화면들도 지속적으로 사용하는 데이터에 대해 설정하면 여러 페이지에서 편리하게 데이터를 사용할 수 있다.
		@ModelAttribute("이름")
		public 객체 메소드() {
			return 객체;
		}

	- Model 객체에서 지정된 이름의 정보를 가져온다.
		@RequestMapping("/경로")
		public String 메소드(@ModelAttribute("이름") 객체 변수명, ... ) {
		
		}

@SessionAttributes({"이름"})
	- 클래스에 부착하는 어노테이션이다.
	- "이름"은 @ModelAttribute("이름")에 설정된 것과 같은 이름을 지정하여, @ModelAttribute("이름")으로 얻어진 정보를 세션에 저장한다.
