홈
	메뉴 - 홈, 회원가입, 로그인 링크만 화면에 표시된다.
	내용 - 사이트 안내가 화면에 표시된다.

회원가입
	메뉴 - 홈, 회원가입, 로그인 링크만 화면에 표시된다.
	내용 - 회원가입 모달창이 화면에 표시된다.
	AJAX
		요청 파라미터 
			- id, name, password, email
		응답처리
			- 성공
				{status:"success", username:"홍길동"}
				모달창에 회원가입 성공 메세지가 표시된다.	
			- 실패 : {status:"fail"}
				모달창에 회원가입 실패 메세지가 표시된다.

로그인
	메뉴 - 홈, 회원가입, 로그인 링크만 화면에 표시된다.
	내용 - 로그인 모달창이 화면에 표시된다.
	AJAX 
		요청 파라미터
			- id, password
		응답처리
			- 성공
				{status:"success", username:"홍길동"}
				모달창을 hide 시킨다.
				메뉴에서 나의 일정, 로그아웃을 show하고, 회원가입, 로그인은 hide시킨다.
			- 실패
				{status:"fail"}
				모달창에 로그인 실패 메세지가 표시된다.

로그아웃
	메뉴 - 홈, 나의 일정, 로그아웃이 표시된다.
	내용 - 사이트 안내창 혹은 나의 일정 목록이 표시되어 있을 것이다.
	AJAX 
		요청 파라미터
			- 없음
		응답처리
			- 성공
				{status:"success"}
				회원가입, 로그인을 show하고, 나의 일정과 로그아웃을 hide시킨다.

나의 일정
	메뉴 - 홈, 나의 일정, 로그아웃이 표시된다.
	내용 - 나의 일정 리스트가 표시된다.
	AJAX 
		요청 파라미터
			- 없음
		응답처리
			- 실패
				{status: "fail"} <-- 로그인 상태가 아닌 경우
				회원가입, 로그인을 show하고, 나의 일정과 로그아웃을 hide시킨다.
				사이트 안내가 화면에 표시된다.
			- 성공
				{status:"success", todos:[{no:1,title:"장보기"....},{},{}]}
				등록된 일정이 화면에 표시된다.

새 일정 등록
	메뉴 - 홈, 나의 일정, 로그아웃이 표시된다.
	내용 - 일정등록 모달창이 표시된다.		
	AJAX 
		요청 파라미터
			- title, content, day
		응답처리
			- 실패
				{status: "fail"} <-- 로그인 상태가 아닌 경우
				회원가입, 로그인을 show하고, 나의 일정과 로그아웃을 hide시킨다.
				사이트 안내가 화면에 표시된다.
			- 성공
				{status:"success", todo:{no:1, title:"장보기", ...}}
				모달창을 hide 시키고, 새 일정을 목록의 가장 앞에 추가한다.
상세보기
	메뉴 - 홈, 나의 일정, 로그아웃이 표시된다.
	내용 - 일정상세 모달창이 표시된다.
	AJAX 
		요청 파라미터
			- todoNo
		응답처리
			- 실패
				{status: "fail"} <-- 로그인 상태가 아닌 경우
				회원가입, 로그인을 show하고, 나의 일정과 로그아웃을 hide시킨다.
				사이트 안내가 화면에 표시된다.
			- 성공
				{status:"success", todo:{no:1, title:"장보기", ...}}
				모달창에 일정상세정보를 표시한다.
