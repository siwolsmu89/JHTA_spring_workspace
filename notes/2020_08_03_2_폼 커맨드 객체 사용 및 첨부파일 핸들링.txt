폼 커맨드 객체
	* 폼 커맨드 객체는 입력필드의 값을 저장하기 위해서 생성한 객체이다.
	* xxxForm 형태의 이름으로 생성한다.
	* <form /> 태그에서 정의한 모든 입력필드 값을 한 번에 담기 위해서 사용한다.
	* 요청핸들러 메소드에서 폼 커맨드 객체를 매개변수로 선언하면, 
	  HandlerAdapter가 
		1. 폼 커맨드 객체를 생성한다. (default 생성자가 꼭 필요하다.)
		2. 폼 커맨드 객체의 필드명을 검색한다.
		3. 폼 커맨드 객체의 필드명과 동일한 이름의 요청파라미터 값을 요청객체에서 조회한다.
		4. 조회된 값을 폼 커맨드 객체의 필드에 저장한다.
		   (기본 자료형 타입, String 타입 등의 경우, 필요하다면 형 변환을 수행한다.)
		5. 모든 요청파라미터 값의 저장이 완료되면, 요청 핸들러 메소드를 실행하면서 매개변수로 폼 커맨드 객체를 전달한다.
	* 요청파라미터로 전달된 파일은 폼 커맨드에서 MultipartFile 타입의 필드를 생성해 저장한다.

폼 커맨드 객체를 매개변수로 선언하고, 사용하기
	* <form /> 태그의 모든 입력필드값을 저장하는 폼 커맨드 객체를 요청 핸들러 메소드의 매개변수로 선언하면, 
	  HandlerAdapter가 요청 파라미터값을 조회해서 폼 커맨드 객체에 자동으로 값을 저장한 후, 요청 핸들러 메소드 실행 시 전달해준다.
	* <form /> 태그의 입력필드 개수가 많을 때 반복적인 코드를 효과적으로 줄일 수 있다.
	* BeanUtils.copyProperties(원본, 대상)
		* 원본 객체의 필드값을 대상 객체의 필드에 복사한다.
		* 원본 객체의 필드명과 대상 객체의 필드명이 동일한 필드만 복사가 발생한다.
		* 원본 객체와 대상 객체의 필드명은 동일하지만, 타입이 서로 다를 때에는 에러가 발생한다.
		* 원본 객체와 대상 객체의 필드를 선택적으로 복사할 수 없다.
		  (필드명은 동일하지만 타입이 다른 필드가 존재하지 않아야 한다.)
		* 원본 자리에 폼 커맨드 객체를, 대상 자리에 VO 객체를 놓으면 매개변수로 받은 폼 커맨드 객체를 VO 객체로 변환할 수 있다.
	* 폼 커맨드 객체를 VO 객체에 복사한 후, 해당 VO 객체를 사용하는 Service 메소드를 호출하여 업무를 처리한다.

첨부파일 핸들링
	MultipartFile
		* 첨부파일 업로드를 지원하는 객체다.
		* <form /> 태그에 enctype="multipart/form-data" 속성이 설정되어 있고, 첨부파일 필드가 있을 때, 
		  폼 커맨드 객체에 첨부파일 입력 필드와 동일한 이름의 필드가 있으면, 첨부파일의 존재 여부와 무관하게 MultipartFile의 값은 NULL이 아니다.
		  첨부파일이 없을 경우에는 해당 MultipartFile 객체에서 isEmpty() 메소드를 실행했을 때 true가 반환된다.
		* 주요 API
			boolean isEmpty()
				해당 필드명으로 업로드된 첨부파일이 없으면 true를 반환한다.
			String getOriginalFileName()
				첨부파일명을 반환한다.
			byte[] getBytes()
				첨부파일의 내용을 byte[] 배열로 반환한다.
			InputStream getInputStream()
				서버의 임시 디렉토리에 저장된 첨부파일을 읽어오는 스트림을 반환한다.
			String getContentType()
				첨부파일의 컨텐츠타입을 MIME 형식으로 반환한다.
			long getSize()
				첨부파일의 크기를 반환한다.

첨부파일 처리하기 절차
	1. 폼 커맨드 객체에서 첨부파일 정보를 가지고 있는 MultipartFile객체를 조회한다.
	2. isEmpty() 메소드를 사용해 업로드된 첨부파일이 존재하는지 확인한다.
	3. 첨부파일이 존재할 경우, 업로드된 파일의 이름을 조회한다.
	4. 파일이름 중복으로 덮어쓰기가 발생하지 않도록 파일 이름에 유닉스타임을 추가한다.
	5. 업로드된 첨부파일을 저장하기 위한 파일 객체를 생성한다.
	6. FileCopyUtils.copy(소스, 대상) 메소드를 사용해 서버의 임시 디렉토리에 저장된 첨부파일을 File 객체가 지정한 위치로 복사한다.
	7. 파일명을 테이블에 저장하기 위해 VO 객체에 파일명을 추가한다.
